# JTL Comparator

Локальное веб-приложение для сравнения результатов нагрузочного тестирования JMeter.

## Возможности

- Загрузка двух `.jtl` файлов с произвольными именами прогонов
- Автоматическое определение Transaction Controller vs HTTP-сэмплеров
- Агрегация метрик: Avg, p50, p90, p95, p99, Min, Max, RPS, Error Rate
- Сравнительная таблица с цветовой подсветкой деградации/улучшения
- Drag & Drop загрузка файлов
- Sticky header при скролле
- Итоговая строка Summary
- Экспорт: Markdown (для Confluence) и CSV

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python app.py
```

Открыть в браузере: http://localhost:5000

## Правила подсветки дельты

### Метрики времени (Avg, p95, p99) — деградация = рост

| Δ% | Цвет |
|---|---|
| > +20% | 🔴 Красный (critical) |
| +10% … +20% | 🟡 Жёлтый (warning) |
| −10% … +10% | Нейтральный |
| < −10% | 🟢 Зелёный (improved) |

### RPS — деградация = снижение

| Δ% | Цвет |
|---|---|
| < −20% | 🔴 Красный |
| −20% … −10% | 🟡 Жёлтый |
| > +10% | 🟢 Зелёный |

### Error Rate

| Условие | Цвет |
|---|---|
| Err R2 > Err R1 + 3% | 🔴 Красный |
| Err R2 > Err R1 + 1% | 🟡 Жёлтый |
| Err R2 < Err R1 | 🟢 Зелёный |
