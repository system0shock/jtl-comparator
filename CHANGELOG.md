# Changelog

Все заметные изменения проекта документируются в этом файле.

Формат основан на Keep a Changelog.

## [0.4.0-alpha.1] - 2026-03-03

### Added
- Первый публичный alpha-релиз.
- Веб-интерфейс сравнения JTL с настраиваемыми правилами подсветки.
- Сортировка таблицы по имени и всем параметрам + нумерация строк.
- Экспорт Markdown/CSV с учетом текущей сортировки.
- API/Unit/E2E (Playwright) тесты.
- Автоматический релизный workflow по тегам `v*`.

## [0.3.0] - 2026-03-03

### Added
- Настраиваемые правила подсветки дельты в UI и прокидывание правил в backend.
- Подробные tooltip'ы для значений дельты с формулой и интерпретацией.
- Справка в UI «Как пользоваться инструментом» (в формате спойлера).
- Новый документ архитектурной карты: `CODE_GRAPH.md`.
- API-тесты `tests/test_app_api.py`.
- Расширенные unit-тесты правил дельты и edge-cases.
- Релизная инфраструктура:
  - `VERSION`
  - `CHANGELOG.md`
  - GitHub Actions workflow для релизов по тегу `v*`.

### Changed
- Обновлена документация в `README.md` по правилам дельты, примерам и работе с интерфейсом.
- Добавлен `.gitignore` для служебных файлов.

## [0.2.0] - 2026-03-03

### Added
- Взвешенный `SUMMARY` по `Samples`.
- Безопасная обработка `Error Rate` при отсутствии транзакции в одном из прогонов.
- Параметры запуска через env (`FLASK_DEBUG`, `HOST`, `PORT`).
- Базовый тестовый набор для `jtl_analyzer`.

## [0.1.0] - 2026-03-03

### Added
- Первая рабочая версия JTL Comparator (Flask + pandas + vanilla JS).
